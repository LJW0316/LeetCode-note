## 题解

### 循环

每次将结果res左移一位再加上n的最后一位，并将n右移一位，重复32次。

时间复杂度O(logn)，空间复杂度O(1)。

```c++
class Solution {
public:
    uint32_t reverseBits(uint32_t n) {
        uint32_t res = 0;
        for (int i = 0; i < 32; i++) {
            res = res << 1 | (n & 1);
            n = n >> 1;
        }
        return res;
    }
};
```

### 分治

将数n的前16位和后16位交换，然后将n每16位中的前8位和后8位交换，再将n中的每8位中方的前4位与后4位交换，再将n中的每4位的前2位和后两位交换，再将n中的每两位的前1位和后1位交换即可。

时间复杂度O(1)，空间复杂度O(1)。

```c++
class Solution {
public:
    uint32_t reverseBits(uint32_t n) {
        n = (n >> 16) | (n << 16);
        n = ((n & 0xff00ff00) >> 8) | ((n & 0x00ff00ff) << 8);
        n = ((n & 0xf0f0f0f0) >> 4) | ((n & 0x0f0f0f0f) << 4);
        n = ((n & 0xcccccccc) >> 2) | ((n & 0x33333333) << 2);
        n = ((n & 0xaaaaaaaa) >> 1) | ((n & 0x55555555) << 1);
        return n;
    }
};
```
