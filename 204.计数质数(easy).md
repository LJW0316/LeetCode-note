## 题目

统计所有小于非负整数 n 的质数的数量。

 

示例 1：

输入：n = 10
输出：4
解释：小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。
示例 2：

输入：n = 0
输出：0
示例 3：

输入：n = 1
输出：0


提示：

0 <= n <= 5 * 106

## 题解

### 暴力法（超时）

如果n<=2直接返回0，对于其他数从3枚举到n对于每个i进行一次质数判断，从2枚举到sqrt(i)。

时间复杂度O(nlogn)，空间复杂度O(1)。

```c++
class Solution {
public:
    int countPrimes(int n) {
        if (n <= 2) {
            return 0;
        }
        int cnt = 1;
        bool flag = true;
        for (int i = 3; i < n; i++) {
            flag = true;
            for (int j = 2;j * j <= i; j++) {
                if (i % j == 0) {
                    flag = false;
                    break;
                }
            }
            if (flag) {
                cnt++;
            }
        }
        return cnt;
    }
};
```
