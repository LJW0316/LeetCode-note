## 题目

给定一个整数，写一个函数来判断它是否是 3 的幂次方。如果是，返回 true ；否则，返回 false 。

整数 n 是 3 的幂次方需满足：存在整数 x 使得 n == 3x

 

示例 1：

输入：n = 27
输出：true
示例 2：

输入：n = 0
输出：false
示例 3：

输入：n = 9
输出：true
示例 4：

输入：n = 45
输出：false


提示：

-2^31 <= n <= 2^31 - 1


进阶：

你能不使用循环或者递归来完成本题吗？

## 题解

### 循环迭代

若n为0直接返回false。对其他的n，如果n除以3余数为0则，不断除以3，直到不能除尽。最后判断n是否为1，为1返回true，否则返回false。

```c++
class Solution {
public:
    bool isPowerOfThree(int n) {
        if (n == 0) {
            return false;
        }
        while (n % 3 == 0) {
            n /= 3;
        }
        return n == 1;
    }
};
```
